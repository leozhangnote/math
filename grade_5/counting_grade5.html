Combinatorial Counting

Grade 5  ·  Multiplication & Addition Principles, Pair Counting, Systematic Enumeration, Partitions, and Maximum Regions

Multiply Add Pairs Enumerate Partition Regions
  • Multiplication Principle: independent sequential stages → multiply all stage counts.
  • Addition Principle: mutually exclusive cases (OR) → add the case counts.
  • Pair counting: n objects, every pair counted once → n×(n−1)÷2. Diagonals of n-gon: n×(n−3)÷2.
  • Bills/values: a bills of type X and b bills of type Y → (a+1)×(b+1)−1 non-zero amounts.
  • Digit restrictions: handle the restricted position first, then multiply remaining free choices.
  • Partitions (each ≥ 1, 3 groups): (n−1)×(n−2)÷2. Each ≥ 2: give 1 to each group first → (n−4)×(n−5)÷2.
  • Max regions (n straight cuts): n×(n+1)÷2 + 1. Each new cut intersects all previous cuts, adding one more region per intersection.
