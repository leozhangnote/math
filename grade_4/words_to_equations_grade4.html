Setting Up an Equation — Key Forms

  • x + a = b — "increased by a", "a added to x"
  • x − a = b — "decreased by a", "minus a"
  • a − x = b — "x subtracted FROM a" ← watch the direction!
  • ax = b — "a times x", "x multiplied by a", "a times as many"
  • x ÷ a = b — "divided by a", "one-ath of x", "shared equally among a"
  • ax + b = c — a groups of x, plus b extra items, equals c total
  • a(x + b) = c — a groups, each with (x + b) items, equals c total

How to Define x

  • Pick the unknown quantity: "Let x = ..."
  • Express every other quantity in terms of x.
  • Find the relationship in the story and write it with an = sign.
  • Check: plug in a trial answer — does it fit the original story?
