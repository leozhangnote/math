Building a System — Key Principle

  • One constraint = one equation. Count the constraints in the story — you need exactly that many equations.
  • One equation per unknown. Two unknowns need two equations; three unknowns need three equations.

Common System Forms

  • Sum & Difference: ① x + y = total   ② x − y = difference
  • Count + Quantity: ① x + y = heads   ② ax + by = total quantity
  • Two Prices: ① a₁x + b₁y = cost₁   ② a₂x + b₂y = cost₂
  • Ratio + Total: ① x = ry (or rx = y)   ② x + y = total
  • Three Variables: need exactly 3 equations for 3 unknowns

Direction Checks

  • "x subtracted FROM a" → a − x (not x − a)
  • "r times as many as y" → x = ry (not y = rx)
  • Coefficient of x = quantity/rate that belongs to the thing you called x
