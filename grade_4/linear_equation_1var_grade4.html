Balance Model — Inverse Operations

  • An equation is like a balance scale — do the same thing to both sides and it stays equal.
  • x + a = b → subtract a: x = b − a
  • x − a = b → add a: x = b + a
  • a − x = b → x = a − b (think: "a minus what gives b?")
  • ax = b → divide by a: x = b ÷ a
  • x ÷ a = b → multiply by a: x = b × a
  • ax + b = c → subtract b first, then divide by a
  • a(x + b) = c → divide by a first, then subtract b
  • Always verify: substitute x back — left side must equal right side.
